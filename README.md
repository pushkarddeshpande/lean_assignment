# Lean_Assignment

# Task # 1 - You are tasked with developing an automated test suite for Sauce labs  demo website. Your automation test suite should cover the customer flow of selecting 3 random items and completing the checkout flow.

# Requirements:

# Test Coverage: Your test suite should cover the scenario of a user completing a successful checkout. You do not need to cover any negative scenarios.
# Test Framework: The framework must be designed using Javascript using automation tool Cypress
# Assertions: Include appropriate assertions to verify the correctness of the user's actions.
# Reporting: Implement reporting mechanisms to provide clear feedback on test results.

# Test Execution 

# Run Tests via CLI & headless browser via scripts
`npm run test`

# Run Tests via CLI & headed browser via scripts
`npm run test-headed`

# Run Tests in Cypress UI Runner via scripts 
`npm run test-ui-runner`

# To Run Mochawesome Report 
`npm run mochawesome-merge`

# To Run Allure Report 
`npm run allure-report`
